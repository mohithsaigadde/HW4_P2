Dynamic Multiplication Table – HW4 Part 2

Enter integer ranges (−50..50). The top row are multipliers (horizontal), the first column are multiplicands (vertical). Sliders and text boxes are synchronized, and you can save different tables into tabs.

Live Preview

The preview table updates automatically when you change any text field or move a slider (as long as the inputs are valid).

Manage Saved Table Tabs

Use the × icon on a tab to delete it individually, or select multiple tables below and delete them all at once.

Tip: Use different ranges and save each result into its own tab to compare tables side by side.
